# Scriptica
> In‑game scripting for Minecraft (Fabric). Automate the boring stuff.

- Client-side scripts you can edit and run in-game
- Events + automation helpers (keys, look/turn, mining/placing)
- A small custom language focused on practical Minecraft tasks
