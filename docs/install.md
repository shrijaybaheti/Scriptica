# Install

Scriptica is a **client-side** Fabric mod for Minecraft **1.21.11**.

## Requirements
- Minecraft Java Edition
- Fabric Loader for **1.21.11**
- Fabric API for **1.21.11**

## Steps
1. Download the latest Scriptica release `.jar` from GitHub Releases.
2. Put the `.jar` in your `.minecraft/mods/` folder.
3. Make sure Fabric API is also in `.minecraft/mods/`.
4. Launch the game.

## Open Scriptica
- Default hotkey: **`** (backtick / grave accent)
- Change it in **Options → Controls → Misc → Open Scriptica**

## Troubleshooting
- If the mod doesn’t show up, confirm your Minecraft/Fabric versions match **1.21.11**.
- If you installed the mod but the hotkey does nothing, check **Options → Controls** for conflicts.
